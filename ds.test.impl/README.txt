В сформировавшейся библиотеке реализован шаблон проектирования Простая Фабрика:

а) интерфейс IPluginFactory задаёт обязательные к реализации параметры для прочих фабрик (создающих объекты)
б) создана реализация PluginFactory как образец
в) в зависимости от поданного на вход параметра в Program.cs PluginFactory инициализирует
	объект из перечня классов, реализующих интерфейс IPlugin
г) каждый из классов семейства IPlugin (Addition, Multiplication, Basic)
	имеет несколько полей, к примеру, версию (Version), описание (Description)
д) статический класс Plugin хранит информацию о количестве плагинов и их условных кодах
